# DBX 패키지

DBX는 다양한 언어와 플랫폼에 대한 공식 바인딩을 제공합니다.

| 패키지 | 레지스트리 | 설치 |
|--------|-----------|------|
| [dbx-core]({{ site.baseurl }}/korean/) | crates.io | `cargo add dbx-core` |
| [DBX.Dotnet]({{ site.baseurl }}/korean/dotnet/) | NuGet | `dotnet add package DBX.Dotnet` |
| [dbx-py]({{ site.baseurl }}/korean/python/) | PyPI | `pip install dbx-py` |
| [dbx-native]({{ site.baseurl }}/korean/nodejs/) | npm | `npm install dbx-native` |
| [C/C++]({{ site.baseurl }}/korean/packages/cpp) | — | 헤더 + DLL/LIB |
